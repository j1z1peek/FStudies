# FStudies - платформа с теорией и практикой по программированию
# https://fstudies-2282h.web.app/

## Функционал

  - Авторизация
  - Изучение теории
  - Прохождение тестов
  - Возможность посмотреть последний результат теста

## Технологии

 - HTML, CSS, JavaScript
 - Firebase для авторизации, хоста и базы данных
